Application Procedure

 
  1. Remplissez le formulaire d'application et envoyez-le par courriel à gcu@gcubureau.org et/ou par la poste à GCU Bureau Sprl, 500, avenue Louise, 1050 Bruxelles, Belgique.
    Vous aurez besoin d'un code RICS valide pour l'application CUU. Vous pouvez demander un code via https://uic.org/rics.
     
    • Remarque: Les frais de contribution annuels 2019 sont de 300 EUR + 0,06 EUR / wagon.
       
  2. Vous recevrez par courriel un login et un mot de passe pour le CUU Broker.
     
  3. Remplissez toutes les données de contact pertinentes dans CUU Broker (un manuel d'utilisation peut être téléchargé ici)
     
  4. En tant que détenteur vous êtes tenu de communiquer les numéros de tous les wagons susceptibles d'être utilisés au titre du CUU. Les numéros de wagon doivent être chargés dans la section banque de données du CUU Broker. Cette opération peut être effectuée manuellement, via une liste Excel, ou via des services Web XML.
     
  5. Après vérification et approbation des informations fournies par le Bureau CUU, l'adhésion sera activée et vous serez inclus dans la liste officielle de signataires à la prochaine publication (cfr. art. 2.4).
